# Bienvenue sur le README de mon inscription

### Instalation

1. Installez XAMPP sur votre machine.
2. Lancez Apache et MySQL depuis le panneau de controle XAMPP.
3. Importez les tables de ma base de donnée en allant dans phpMyAdmin.
4. Récuperez le dossier de mon projet en utilisant le bouton "Code" et choisissez "Download ZIP".

### Utilisation

1. Ouvrez le dossier de mon projet dans votre éditeur de texte préféré.
2. Modifiez le fichier "database.php" pour ajouter votre propre configuration de base de donnée.
   - Remplacez "localhost" par l'adresse de votre serveur MySQL.
   - Remplacez "root" par votre nom d'utilisateur MySQL.
   - Remplacez "" par votre mot de passe MySQL.
   - Remplacez "db" par le nom de votre base de donnée.
3. Ouvrez le fichier "index.php" dans votre navigateur en utilisant le serveur Apache XAMPP.
4. Vous pouvez maintenant vous inscrire en remplissant le formulaire.


# Welcome to my registration README

### Installation

1. Install XAMPP on your machine.
2. Launch Apache and MySQL from the XAMPP control panel.
3. Import my database tables into phpMyAdmin.
4. Download my project folder by clicking the "Code" button and selecting "Download ZIP".

### Usage

1. Open the project folder in your preferred text editor.
2. Modify the "database.php" file to add your own database configuration.
   - Replace "localhost" with your MySQL server address.
   - Replace "root" with your MySQL username.
   - Replace "" with your MySQL password.
   - Replace "db" with the name of your database.
3. Open the "index.php" file in your browser using the XAMPP Apache server.
4. You can now register by filling out the form.
